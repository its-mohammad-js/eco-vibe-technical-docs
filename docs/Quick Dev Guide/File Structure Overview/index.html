File Structure Overview

Main Folder's​

This section provides a brief overview of the project's file structure. While not exhaustive, it offers a general understanding to help you better grasp the overall architecture of the project. Apart from the configuration files located in the root directory, each folder is listed below with its name and a brief description.

  • github: Contains main actions for the project, including deployment and removal of outdated data from the database.
  • public: Stores static and frequently used data for the application.
  • src:
    • common: Holds reusable code, components, hooks, and utility functions for executing certain GitHub actions.
    • config: Stores configuration files for the application.
    • layout: Contains the main layout components, which are configured here.
    • pages: Houses the pages for each route, grouped according to the route structure, along with their corresponding components.
    • reducers: Manages the application state and global functions related to Redux.
    • routes: Contains the primary route groupings for the application.
